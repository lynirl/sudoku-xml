## Représentation des données : Projet Sudoku

Ce projet rassemble les feuilles de styles permettant notre travail sur le projet Sudoku. 

### Transormations XSLT

Le dossier XSLT contient les transormations qui créent des configuration d'une grille de Sudoku : grille gagnante, grille correcte, grille incorrecte, etc...

## Exemples de grilles

Le dossier exemplesGrilles contient des exemples de grilles pour différents cas possibles pour une grille Sudoku : 

- Exemple1: grille remplie incorrecte
- Exemple2: grille remplie correcte/gagnante
- Exemple3: grille incomplète correcte
- Exemple4: grille de base

## Tests pour XSD 

Le dossier XMLsTestsXSD contient des exemples de grilles pour valider ou relever des erreurs à la suite de la création du ```sudoku_schema.xsd```.
